# Аудит и контроль

CloudPub автоматически отслеживает и записывает все действия пользователей для обеспечения безопасности и соответствия требованиям.

## Типы отслеживаемых действий

### Операции с данными
- **Создание** - новые публикации, домены, пользователи
- **Обновление** - изменение настроек
- **Удаление** - удаление объектов

### Аутентификация
- **Вход в систему** - успешные авторизации
- **Ошибки входа** - неудачные попытки
- **Выход из системы** - завершение сеансов

### Подключения к бастиону
- **Подключение** - успешные соединения с RDP/VNC
- **Отключение** - завершение сеансов
- **Ошибки** - неудачные попытки подключения

## Информация в журнале

Каждая запись содержит:
- **Время** - дата и время действия
- **Пользователь** - кто выполнил действие
- **IP-адрес** - откуда выполнено
- **Объект** - над чем выполнялось действие
- **Детали** - подробности операции

## Запись RDP сеансов

Все подключения через бастион могут записываться:

1. Видео всех действий пользователя
2. Конвертация в MP4 для просмотра
3. Скачивание записей для архива

## Права доступа

- **Суперпользователи** - доступ ко всем записям
- **Руководители команд** - записи своей команды
- **Пользователи** - только свои записи

## Экспорт данных

Журнал аудита можно экспортировать:
- **CSV** - для анализа в Excel
- **JSON** - для программной обработки
- **Print** - для печати

## Практическое применение

### Расследование инцидентов
Выясните кто, когда и что сделал в системе.

### Мониторинг безопасности
Отслеживайте подозрительную активность и неудачные попытки входа.

### Соответствие требованиям
Полная история для аудиторов и проверок безопасности.
