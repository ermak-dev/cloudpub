# Двухфакторная аутентификация

CloudPub поддерживает **двухфакторную аутентификацию (2FA)** для максимальной защиты вашей учетной записи и опубликованных сервисов.

## Зачем нужна 2FA

Даже если злоумышленник узнает ваш пароль, он не сможет войти без доступа к вашему устройству с приложением-аутентификатором.

## Поддерживаемые приложения

- **Яндекс Ключ** (iOS, Android)
- **Google Authenticator** (iOS, Android)
- Любое TOTP-совместимое приложение

## Включение 2FA

1. Откройте **Настройки** в личном кабинете
2. Перейдите на вкладку **"Второй фактор"**
3. Нажмите **"Включить 2FA"**
4. Отсканируйте QR-код приложением
5. Введите код подтверждения
6. **Сохраните резервные коды!**

## Вход с 2FA

1. Введите email и пароль
2. Откройте приложение-аутентификатор
3. Введите 6-значный код
4. Код обновляется каждые 30 секунд

## Резервные коды

- Сохраните в надежном месте
- Каждый код используется только один раз
- Нужны при потере устройства

## 2FA в бастионе

При подключении к RDP/VNC серверам через бастион также требуется ввести код 2FA, обеспечивая дополнительную защиту удаленного доступа.

## Преимущества

- **Защита от взлома** - пароль недостаточен для входа
- **Контроль доступа** - уведомление о попытках входа
- **Соответствие стандартам** - требование для критичных систем
- **Работает офлайн** - не требует интернета на телефоне
