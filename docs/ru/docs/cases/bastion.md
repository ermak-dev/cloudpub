# RDP и VNC бастион

CloudPub предоставляет **централизованный доступ к RDP и VNC серверам** с полным контролем, аудитом и записью всех действий пользователей.

## Возможности бастиона

- **Запись сеансов** - видео всех действий для последующего анализа
- **Централизованная аутентификация** - вход через учетные данные CloudPub
- **Двухфакторная аутентификация** - дополнительный уровень защиты
- **Детальный аудит** - полная история подключений

## Публикация RDP сервера

```bash
# Публикация локального RDP
clo publish rdp 3389

# Публикация RDP в локальной сети
clo publish rdp 192.168.1.100:3389

# С указанием имени
clo publish rdp 3389 --name "Рабочий компьютер"
```

## Публикация VNC сервера

```bash
# Публикация локального VNC
clo publish vnc 5900

# Публикация VNC в локальной сети
clo publish vnc 192.168.1.50:5900
```

## Подключение к серверу

1. Подключитесь к **rdp.cloudpub.ru** стандартным RDP клиентом
2. Введите учетные данные CloudPub
3. Введите код 2FA (если включена)
4. Выберите сервер из списка

## Просмотр записей

1. Откройте раздел **"Аудит"** в личном кабинете
2. Найдите нужный сеанс подключения
3. Перейдите на вкладку **"Видео"** для просмотра записи

## Преимущества

- **Единая точка входа** - все серверы в одном месте
- **Безопасность** - шифрование и 2FA
- **Контроль** - запись и аудит всех действий
- **Простота** - не требуется настройка роутера
